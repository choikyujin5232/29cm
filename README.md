# 29CM Web Redesign

> 기존 29CM 메인 페이지를 분석하고, **UI/UX 개선**을 목표로 리디자인 및 퍼블리싱한 프로젝트입니다.

- **배포**: [https://choikyujin5232.github.io/29cm/](https://choikyujin5232.github.io/29cm/)
- **기술 스택**: HTML5, CSS3, JavaScript, Swiper.js
- **주요 기능**:
  - PC/모바일 반응형 레이아웃
  - Swiper.js 배너 슬라이드
  - 스크롤 시 애니메이션(IntersectionObserver)
